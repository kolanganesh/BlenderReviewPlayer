## Why BlenderReviewPlayer Was Created

In many animation and production environments, artists, supervisors, and clients often face difficulties reviewing animation outputs across different platforms. Most review tools require complex setup processes, user logins, or expensive licensed software, making quick review and feedback workflows inefficient.

BlenderReviewPlayer was created to solve these challenges by providing a lightweight, cross-platform review solution for both Windows and Linux users.

The primary goal of BlenderReviewPlayer is to allow end users to quickly access and review their respective production tasks without requiring any login process or complicated pipeline setup. Users can directly open and review assigned assets, animation playblasts, TT renders, and final animation movies in a simple and efficient interface.

---

# Key Features

## Cross-Platform Support
BlenderReviewPlayer is designed to work seamlessly on both Windows and Linux environments, ensuring compatibility across studio production pipelines.

## No Login Required
Artists and reviewers can instantly access their review content without authentication barriers, making the review process faster and more user-friendly.

## Asset and Animation Review
The player supports reviewing:

- Production assets
- Animation TT renders
- Playblasts
- Final animation movies
- Task-based review content

---

# Advanced Compare Modes

To improve review quality and visual analysis, BlenderReviewPlayer includes multiple comparison modes such as:

## Side by Side Mode
Compare two versions simultaneously for easier shot validation and revisions.

## Difference Mode
Highlight visual differences between renders or animation versions for accurate quality checking.

## A/B Compare Mode
Quickly switch between two versions to identify animation or render changes efficiently.
